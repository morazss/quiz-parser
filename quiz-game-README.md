# Quiz Game (CSV Quiz)

Консольная программа на Go, которая загружает вопросы и ответы из CSV-файла и проводит тест-квиз с ограничением по времени.

## Возможности

- Чтение вопросов/ответов из CSV-файла в формате `question,answer`
- Ограничение по времени на весь квиз (по умолчанию 30 секунд)
- Подсчёт количества правильных ответов
- Гибкая настройка через флаги командной строки

## Установка

```bash
git clone <ссылка-на-репозиторий>
cd <папка-проекта>
go build -o quiz main.go
```

## Использование

```bash
./quiz -csv=problems.csv -limit=30
```

### Флаги

| Флаг     | По умолчанию     | Описание                                      |
|----------|-------------------|------------------------------------------------|
| `-csv`   | `problems.csv`    | Путь к CSV-файлу в формате `question,answer`  |
| `-limit` | `30`              | Лимит времени на квиз в секундах              |

## Формат CSV-файла

```csv
5+5,10
7+3,10
1+1,2
8+3,11
```

Первая колонка — вопрос, вторая — правильный ответ. Пробелы вокруг ответа обрезаются автоматически.

## Как это работает

1. Программа читает и парсит CSV-файл в список структур `problem`.
2. Запускается таймер на заданное количество секунд.
3. Пользователю последовательно задаются вопросы; ввод ответа читается в отдельной goroutine.
4. Если время вышло до того, как пользователь успел ответить на все вопросы — квиз останавливается досрочно.
5. В конце выводится итоговый результат: количество правильных ответов из общего числа вопросов.

## Возможные улучшения

- Перемешивание вопросов (`-shuffle` флаг)
- Поддержка UTF-8 без учёта регистра в ответах
- Цветной вывод результатов
